Jeu du Pendu



Le Pendu est un jeu consistant à trouver un mot en devinant quelles sont les lettres qui le composent. Le but du jeu est simple : deviner toute les lettres qui doivent composer un mot, éventuellement avec un nombre limité de tentatives. A chaque fois que le joueur devine une lettre, celle-ci est affichée. Dans le cas contraire, le dessin d’un pendu se met à apparaître.



Mode utilisateur :

Dans ce mode, l'utilisateur doit deviner un mot que l'ordinateur propose.

Mode Ordinateur :

Dans ce mode l'utilisateur propose un mot et l'ordinateur essaie de le divinez.
